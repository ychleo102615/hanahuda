現代日式簡約風格 (Modern Japanese Minimalist)
整體採用簡約的排版 (大量留白、清晰的字體) 搭配精緻的花牌視覺元素，以深色背景或淡雅色調襯托花牌的鮮豔色彩。

|CTA 按鈕|	開始遊戲 可在導航列上以高對比色的按鈕形式呈現，與其他連結區分。|確保點擊區域足夠大，特別是在移動裝置上。

# 3. 規則說明內容 (Rules Section)

這是內容最豐富的部分，應採用清晰的卡片式排版 (Card Layout) 或分步流程圖 (Step-by-step Flow)。

|元素|	設計建議|	互動與響應式特點
|-|-|-|
|版面結構|	採用兩欄或三欄網格排版，每張卡片代表一個主題 (目標、牌組、役種等)。|	在手機上自動堆疊成單欄，確保內容不擁擠。
|役種介紹	|使用一個可水平滑動的輪播圖 (Carousel)，每張圖展示一個經典役種 (如五光、四光)，圖片為役種的牌組，配上簡潔的名稱和得分說明。|	折疊/展開狀態 (Accordion/Toggle)：將「遊戲目標」、「牌組構成」等基本內容預設展開。將「基本役種介紹」設為可折疊的區塊，使用者點擊役種名稱時才展開詳細說明和圖例。
視覺呈現|	搭配精選花牌圖像作為圖標或區塊背景，直觀展示規則內容。|	確保所有圖例在不同尺寸下都能清晰呈現。

# 💡 整體風格與色彩建議

## 色彩計畫：
- 主色調 (Primary)： 採用深海軍藍、墨黑色或深木紋色，營造現代、沉穩的基調。
- 輔助色 (Accent)： 採用花牌中常見的鮮紅色 (如光牌上的太陽)、櫻粉色或亮綠色 (如草牌的綠色)，用於 CTA 按鈕和重要連結，形成高對比，吸引目光。
- 文字色： 高對比的白色或淺灰色。
## 響應式設計： 
必須採用 Flexbox 或 Grid 排版，確保無論在手機、平板或桌機上，版面都能完美適應，這是現代網站的基礎。
## 微互動： 
在頁面捲動時，讓花牌圖案或文字塊漸入 (Fade In) 或輕微滑入 (Slide Up) 螢幕，增加網站的動態感和精緻度。
