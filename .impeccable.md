## Design Context

### Users
日本傳統花札愛好者與桌遊玩家，在休閒或競技場合下進行線上對局。使用者的核心任務是：專注比賽、判讀手牌、做出策略決策、感受每一局的起伏。他們希望介面「消失」在遊戲體驗之中，不因 UI 干擾而分心。

### Brand Personality
**典雅 · 沉穩 · 精緻**

語調沉靜而不冷漠；精準而不繁瑣；有底蘊而不過度裝飾。如同一張鋪展在桌上的深綠色絨布，一切都恰得其分。

情緒目標：讓玩家在完成役種、贏得對局時，感受到**滿足與成就**——那種低調但真實的悅感，而非過度誇張的慶祝動畫。

### Aesthetic Direction
**金箔蒔絵 × 現代極簡**（Kinpaku Maki-e meets Quiet Luxury）

- **色彩故事**：深墨色底（#0d1f17, #111827）搭配克制的金色（#D4AF37, #B8860B）點綴，主色調為冷藍灰系（primary-500 to primary-800）。彩色只出現在功能性語意：紅色 CTA、綠色成功、橘色警示。
- **視覺語言**：日本傳統工藝紋樣（青海波、麻の葉）作為背景肌理，不喧賓奪主；金箔框線（kinpaku-frame）為遊戲區域帶來儀式感。
- **光影質感**：玻璃擬態（glassmorphism）用於導覽與 Modal，保持資訊層次清晰。深色層疊，輕量 backdrop-blur。
- **動態節奏**：動畫以 ease-in-out 為主調，時長 200ms–600ms。避免彈跳或過度動態；動作服務於資訊傳達，而非娛樂表演。
- **主題**：深色模式唯一，無需淺色版本。
- **參考基準**：現有設計系統已成熟，所有新設計應與既有風格語言保持一致，不引入衝突的視覺方向。

### Design Principles

1. **介面即退場** — UI 的最高境界是讓玩家忘記它的存在。減少視覺噪音，讓花札牌面與遊戲資訊成為畫面主角。

2. **克制的裝飾性** — 金色與花紋是點綴，不是主角。每一個裝飾元素都應有語意理由，避免「因為好看而加」。

3. **成就感的精確傳達** — 動畫與視覺回饋的目的是強化成就感（役種達成、勝局），而非製造噪音。每個慶祝動效都應感覺被賺到，而非理所當然。

4. **一致性優於創意** — 擴充新元件時，優先從現有設計語言（kinpaku-frame、glass topbar、gold-border tokens）取材，而非引入新的視覺風格。設計系統的生命在於一致。

5. **WCAG AA 為底線** — 色彩對比度、鍵盤可操作性、focus-visible 狀態為必要條件，不因遊戲美感而妥協。
